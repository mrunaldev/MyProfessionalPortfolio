## 🚀 How to Use

### Option 1: Local Development
1. Clone or download this repository
2. Open `index.html` in your web browser
3. No server required - it's a static website

### Option 2: Deploy to GitHub Pages
1. Create a GitHub repository named `mrunal-portfolio-website`
2. Push all files to the repository
3. Go to repository Settings → Pages
4. Set source to "main branch"
5. Access your site at `https://yourusername.github.io/mrunal-portfolio-website`

### Option 3: Deploy to Other Hosting
- **Netlify**: Drag and drop the folder
- **Vercel**: Connect your GitHub repository
- **AWS S3 + CloudFront**: Upload files to S3 bucket
- **Heroku**: Use free tier static hosting
- **Firebase Hosting**: Simple deployment with CLI
